## 🔧 TROUBLESHOOTING

### Issue: Dashboard looks plain
→ Clear browser cache: Ctrl+Shift+Delete → Clear cache → Refresh

### Issue: Animations not smooth
→ Check browser hardware acceleration (should be ON)
→ Try Chrome/Edge (better performance)

### Issue: Colors look wrong
→ Verify Flask is serving latest build
→ Check backend/static/ has all files
→ Restart Flask backend

### Issue: Sidebar buttons not clickable
→ Ensure latest build deployed
→ Check browser console for errors (F12)
